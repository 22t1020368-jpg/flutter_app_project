## Công nghệ sử dụng

- Dart

## Hướng dẫn chạy
1. Cài đặt Flutter SDK
2. Clone project từ GitHub
3. Chạy lệnh `flutter pub get`
4. Chạy ứng dụng bằng lệnh `flutter run`
